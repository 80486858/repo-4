Easy ad code management.

== Description ==

Easily manage your ads via nice and simple UI. Each ad code has unlimited amount of conditionals. Conditionals are core WordPress
methods like is_page(), is_category(), or your custom methods. 

Currently works with:

*	Doubleclick for publishers
This is the long description.  No limit, and you can use Markdown (as well as in the following sections).
*	More to come
